# Задание 1

### Текст задачи

Дано натуральное число n. Найти сумму первой и последней цифры этого
числа. 

### Алгоритм решения

1.запрашиваем у пользователя число (именно число а не цифру) с точкой в конце
2. читаем это число
3. передаем в функцию сумма чисел само число и пустую (необъявленную) переменную 
4. в этой функции вызываем две других функции для поиска первого и последнего числа 
5. в поиске первой просто рекурсивно делим число пока оно не станет цифрой и возвращаем итог
6. в поиске крайнего значения просто делим с остатком на 10 и получаем крайнее число 
7. суммируем и возвращаем результат и выводим

# Задание 2

### Текст задачи

Сколько раз входит заданный элемент в дерево?

### Алгоритм решения
1.запрашиваем у пользователя шаблонный список с точкой в конце
2. читаем список
3. передаем в функцию список и пустую переменную 
4. эта функция вызывает встроенную функцию инклуд (как мап) и она создает список из нечетных чисел 
5. после новый список из нечетных элементов проверяется если он не пустой то передается на встроенную функцию сортировки у него забирается только первый элемент
6. возвращаем и выводим
